    • Projects

    Movie Recommendation System

    A movie recommendation system using combination of twitter sentiment analysis and content based filter. Trained model was deployed on Google Cloud App Engine and Flask REST API was used to connect requests from android app. I designed basic functionality of login and signup using firebase and saved the user's metadata on cloud bucket.

    PNR Enquiry Android App

    Android App created with help of Indian railways API to provide PNR status of booked ticket. API request returns result in JSON format which is then used to extract result. Additional Android Spin-Kit library was used to display loading animation.

    Sentiment Analysis model on short text type data

    Sentiment classifier with RNN LSTM and deployed model on Heroku cloud. Initially a Naive Bayes Classifier was used but LSTM RNN performed better in this case hence that was chosen.

    Genetic Algorithm to solve travelling salesman problem

    C++ implementation of GA for TSP problem. Evolutionary algorithms are used to get near optimum results by using multiple random search spaces and evolving the ones with best fitness. The algorithm implemented reached some better results for large number of vertices.

    Socket Programming in Java

    Multithreaded server, RMI and other implementations of socket programming in Java. There multiple programs to demonstrate the socket programming concept and concepts of RMIs.

    Deep Learning Notebooks
